# rym-hide-ratings

---

Firefox extension that hides rateyourmusic album ratings until you've rated the album yourself, to stop your rating being influenced by the general consensus.

This works on release pages and artist pages, and only works if you are logged in.

### On release pages

- Release rating and ranks are hidden if you haven't rated the release, and can be toggled.
- If you have rated the release, release ratings will be visible.
- Track ratings are hidden and unbolded, and can be toggled.

### On artist pages

- Track ratings are hidden and unbolded, and can be toggled.
- Release ratings are only shown for releases you have rated, can also be toggled.
